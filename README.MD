Yuki
====
Analyze all videos in the drag and drop folder, export thumbnails and export 
to Excel file.

![icon](https://github.com/loonghao/Yuki/blob/master/src/yuki/resources/images/yuki.png)

Features:
- Analyze videos.
- Export thumbnails.
- Save to Excel.

Request:
- fbs==0.8.6
- PySide2==5.12.2
- XlsxWriter==1.0.2
